柏浪涛法考：贪污贿赂罪

发表于 2024-07-09 更新于 2024-07-12 分类于 刑法学 本文字数： 5.8k 阅读时长 ≈ 5 分钟

本文是柏浪涛 2024 年刑法法考精讲卷的贪污贿赂部分笔记。

一、贪污罪

（一）行为

第三百八十二条【贪污罪】
国家工作人员利用职务上的便利，侵吞、窃取、骗取或者以其他手段非法占有公共财物的，是贪污罪。
受国家机关、国有公司、企业、事业单位、人民团体委托管理、经营国有财产的人员，利用职务上的便利，侵吞、窃取、骗取或者以其他手段非法占有国有财物的，以贪污论。
与前两款所列人员勾结，伙同贪污的，以共犯论处。

贪污罪本质是财产犯罪

  • 实行行为：侵占、盗窃、诈骗
  • 行为主体：国家工作人员（公务说）
  • 行为对象：公共财物（包括财产性利益，如债权）
  • 行为方式：利用职务便利
  • 案例一
    国有企业老板甲与私人企业老板乙合谋，约定国有企业借款 100 万元给私人企业，事后乙将 100 万元中的 60 万元返还给国有企业老板甲，自己得到 40 万元，甲将账户平账。
    此时甲贪污应收账款（财产性利益）100 万元。因为国有企业损失 100 万元，此时甲获得 60 万元，乙获得 40 万元，两人共同犯罪的犯罪所得数额为 100 万元。要牢记“等号原理”，即财产损失数额应当与犯罪所得数额相等。

（二）实行行为类型

  1. 侵占：将国家所有，自己基于某种职务而占有的财物变为自己所有。
  • 案例二
    甲作为国企出纳保管保险箱的备用金，自己有钥匙。甲想要侵占保险金，但不用钥匙而直接撬开保险箱，拿走保险金。
    此时甲依然构成贪污罪。因为侵占罪的占有状态本身是基于职务便利，所以此时再变成所有时不要求利用职务便利。
  1. 盗窃：将他人国家占有的财物，通过平和手段转移为自己占有。
  • 案例三
    国有企业保险箱由国家工作人员甲乙两个人共同保管，甲保管钥匙，乙保管密码。甲通过窃得密码的方式获得了密码，进而取走了财物。
    甲此时侵犯了共同占有人乙的占有，转移为自己所有，所以甲的实行行为属于盗窃，在此基础上构成贪污罪。
  1. 诈骗：利用职务便利，使得对方基于认识错误而处分财物。
  • 案例四
    国有企业经理甲负责管理员工工资，伪造工资表格使得会计乙受到欺骗，进而处分了财物，甲“吃空饷”。
    甲构成骗取型贪污罪。因为是利用了职权让会计受骗。
  • 案例五
    国有企业科员甲出差后伪造了发票，进而获得了报销财物。
    甲仅构成诈骗罪，因为甲在基于认识错误处分财物的时候没有利用自己的职权。

（三）责任

非法占有目的，同时包括自己占有与他人占有。

  • 案例六
    甲不符合低保户身份，民政局官员乙做了虚假的甲的低保户身份，进而甲骗取社保局工作人员丙取得了财产。
    甲构成诈骗罪，乙构成诈骗罪的共犯。由于乙并未实施诈骗罪的实行行为而仅仅是帮助行为，故不能构成贪污罪。
  • 案例七
    甲与民政局局长乙商量，由甲提供危房补助款材料上报民政局，乙直接将危房补助款批给甲，并未分赃。
    乙构成贪污罪的实行犯，甲构成贪污罪的帮助犯。

二、挪用公款罪

第三百八十四条
【挪用公款罪】国家工作人员利用职务上的便利，挪用公款归个人使用，进行非法活动的，或者挪用公款数额较大、进行营利活动的，或者挪用公款数额较大、超过三个月未还的，是挪用公款罪，处五年以下有期徒刑或者拘役；情节严重的，处五年以上有期徒刑。挪用公款数额巨大不退还的，处十年以上有期徒刑或者无期徒刑。
挪用用于救灾、抢险、防汛、优抚、扶贫、移民、救济款物归个人使用的，从重处罚。

（一）行为

实行行为：挪出行为1
客观处罚条件：归个人使用行为（非法活动、营利活动、一般活动超过三个月未归还）

（二）客观处罚条件：挪作私用

个人落了人情：例如以个人名义将单位钱款借给他人。
个人落了好处：例如以单位名义借款，个人获得好处。

（三）挪用公款与贪污罪的区分

没有非法占有目的中的排除意思，即“想还”。

  • 案例一
    甲挪用公款 100 万元，已经构成挪用公款罪（已经具有实际处罚条件）。随后甲出国逃离。
    甲构成贪污罪，无需数罪并罚，可以理解为贪污罪吸收挪用公款罪。
  • 案例二
    甲已经挪用 100 万元，携带 80 万元潜逃。
    甲构成 80 万的贪污罪与 20 万的挪用公款罪。

（四）共同犯罪模型

  • 案例三
    甲为了贩毒挪出公款，乙参与贩毒。
    因为乙没有挪用公款罪的实行行为（挪用行为），乙仅仅构成贩卖毒品罪。甲构成贩卖毒品罪与挪用公款罪。
  • 案例四
    乙为了贩毒指使甲挪用公款。最终既遂。
    甲构成挪用公款罪实行犯与贩卖毒品罪的帮助犯。乙构成挪用公款教唆犯与贩卖毒品实行犯。
  • 案例五
    乙谎称看病需要钱，欺骗甲挪用公款，但是乙实际上拿去贩毒，两周后归还。
    甲构成挪用公款罪的实行犯，具有挪用公款罪的实行行为，客观处罚条件都是故意行为，也要求主客观相一致，所以此时甲并不满足客观处罚条件，甲不构成挪用公款罪。甲没有主观故意故不构成贩卖毒品罪的帮助犯。乙构成贩卖毒品罪实行犯，由于挪用公款罪是真正身份犯，而身份犯的正犯必须有身份，所以乙不能构成挪用公款罪的间接正犯。而由于甲具有挪用公款罪的实行行为，所以由于有正犯构成要件的行为，乙依然可以构成挪用公款罪的教唆犯。

三、受贿罪

第三百八十五条【受贿罪】
国家工作人员利用职务上的便利，索取他人财物的，或者非法收受他人财物，为他人谋取利益的，是受贿罪。
国家工作人员在经济往来中，违反国家规定，收受各种名义的回扣、手续费，归个人所有的，以受贿论处。

（一）行为

  1. 受贿罪的本质：权钱交易，将自己的职务行为与他人的财物之间形成不正当的对价关系。此处的财产包括财产性利益。此处收钱既包括为自己收钱也包括为他人收钱。
  2. 交易方式：
    • 索贿：只要实施索取财物行为就成立受贿罪。实际收到钱时既遂。
    • 被动收受：请托人提出送钱，官员许诺办事，就成立受贿罪。实际收到钱时既遂。
    • 事后受贿：在职务行为时未受贿，在职务行为后收受贿赂。2
  • 案例一
    老板甲给官员乙送钱，官员乙通知甲将钱送给丙，并明确将钱交给丙，丙收下。
    乙首先构成受贿罪既遂。丙根据共犯原理，构成受贿罪共犯。
  • 案例二
    官员甲即将退休时，承诺帮行贿人办事，在退休后收款。
    甲构成受贿罪，因为受贿罪的实行行为是权钱交易行为，甲“订立交易”时已经成立受贿罪实行行为，故成立受贿罪。
  • 案例三
    官员甲在职时正常完成交易，在退休后行贿人给予官员财物。
    甲不构成受贿罪，因为“订立交易”的时候甲并不是国家工作人员了。离职收钱是否构成受贿罪关键在于在职时是否订立权钱交易。
  1. 变相行贿受贿：付定金购买物品事后不要定金；赌博博彩行为故意赔钱等。
  2. 受贿罪既遂标准：官员接受财物，建立事实上占有。如何处分不影响既遂。占有是主客观相一致的行为。不要求民法上取得所有权。
  • 案例四
    官员甲收受了行贿人乙的银行卡，还没取钱即被抓。
    官员甲已经事实上占有了银行卡，受贿罪已经既遂。此时行贿人乙构成盗窃罪与行贿罪的并罚。

（二）数额

第三百八十六条【受贿罪的处罚规定】
对犯受贿罪的，根据受贿所得数额及情节，依照本法第三百八十三条的规定处罚。索贿的从重处罚。

数额是主客观相一致的

  • 定罪数额 3 万（数额较大）
  • 数额巨大 20 万
  • 数额特别巨大 300 万。

（三）罪数

  1. 与渎职罪关系：原则上收钱行为与办事行为是两个行为，故数罪并罚。司法工作人员受贿罪与四个渎职犯罪牵连犯择一重罪。

第三百九十九条
【徇私枉法罪】司法工作人员徇私枉法、徇情枉法，对明知是无罪的人而使他受追诉、对明知是有罪的人而故意包庇不使他受追诉，或者在刑事审判活动中故意违背事实和法律作枉法裁判的，处五年以下有期徒刑或者拘役；情节严重的，处五年以上十年以下有期徒刑；情节特别严重的，处十年以上有期徒刑。
【民事、行政枉法裁判罪】在民事、行政审判活动中故意违背事实和法律作枉法裁判，情节严重的，处五年以下有期徒刑或者拘役；情节特别严重的，处五年以上十年以下有期徒刑。
【执行判决、裁定失职罪】【执行判决、裁定滥用职权罪】在执行判决、裁定活动中，严重不负责任或者滥用职权，不依法采取诉讼保全措施、不履行法定执行职责，或者违法采取诉讼保全措施、强制执行措施，致使当事人或者其他人的利益遭受重大损失的，处五年以下有期徒刑或者拘役；致使当事人或者其他人的利益遭受特别重大损失的，处五年以上十年以下有期徒刑。
司法工作人员收受贿赂，有前三款行为的，同时又构成本法第三百八十五条规定之罪的，依照处罚较重的规定定罪处罚。

  1. 与贪污罪关系：看钱款来源属性是否为公款。
  • 案例五
    甲向民政局局长乙申请补助，乙对甲批补助 10 万元后要求返还 5 万元。
    此时甲属于受贿罪，因为申请的补助已经属于乙。
  • 案例六
    甲向民政局局长乙申请补助，甲要求乙多申请 5 万元，并收下。
    此时甲属于贪污罪，因为申请的多余的钱属于公款故属于“变相贪污”。
  • 案例七
    甲向民政局局长乙行贿申请补助，乙违规发放补助。
    乙违规贪污为甲非法占有，构成贪污罪。甲乙两人构成贪污的共犯。与受贿行贿数罪并罚。
  • 案例八
    司机甲超载，应交罚款 1000 元。司机给交警个人 500 元，交警放行。
    司机属于行贿，交警属于受贿 500 元。同时，国家有应收账款未收上，具有财产损失，此时交警对国家同时构成 1000 元数额贪污罪，司机也是贪污罪的共犯。但是不需要并罚因为本质上都是应该给国家的款项，两个行为紧密联系，重罪贪污罪 1000 元吸收轻罪即可。

四、行贿罪

第三百八十九条【行贿罪】
为谋取不正当利益，给予国家工作人员以财物的，是行贿罪。
在经济往来中，违反国家规定，给予国家工作人员以财物，数额较大的，或者违反国家规定，给予国家工作人员以各种名义的回扣、手续费的，以行贿论处。
因被勒索给予国家工作人员以财物，没有获得不正当利益的，不是行贿。

（一）行为

  1. 主动给钱。要求为了获取不正当利益，给予财物，构成行贿罪，不要求事情办成。
  2. 被动给钱。要求实际获得不正当利益。没有获得利益的不构成行贿罪。3
  3. 为谋取不正当利益。包括非法利益以及其他不正当利益（如谋取不公平的竞争优势）等。

（二）与受贿罪的关系

行贿罪与受贿罪虽然是对象犯，但是两者之间并没有必然的联动关系。

五、多于二人的职务犯罪

（一）斡旋受贿

第三百八十八条【斡旋受贿犯罪】
国家工作人员利用本人职权或者地位形成的便利条件，通过其他国家工作人员职务上的行为，为请托人谋取不正当利益，索取请托人财物或者收受请托人财物的，以受贿论处。

请托人 A 向中间人 B 送钱，中间人通过斡旋行为对终端的官员 C 行贿最终办事。对斡旋受贿而言，A 是买的中间人 B 的权力，权钱交易仅仅发生在 A 与 B 之间，此时 B 只需要许诺给 A 办事即成立斡旋受贿，至于时候是否实施行为或者是否获取到利益在所不问。

斡旋受贿与普通受贿的关系：

  • 中间人如果对最终官员具有直接管理、隶属、制约关系，如“上级找下级”，则中间人只构成普通受贿。
  • 中间人如果仅仅只能“斡旋”于其中，包括“平级”之间或者“下级找上级”，则构成斡旋受贿罪。
  • 案例一
    A 找到财政局长 B 并给付 10 万元，请托向税务局长 C 减税。但是 B 在向 C 请托时并未说收钱事项。C 未收钱。
    B 首先构成斡旋受贿罪。C 不构成受贿罪。
    C 没有收钱但是知情 B 收钱的事实，此时由于 C 是知道 B 收钱故属于帮助 B 收钱，属于帮助他人收钱，此时 B 同时构成斡旋受贿罪与受贿罪，C 构成受贿罪。
    C 收了 B 的钱，C 构成受贿罪，B 同时构成 C 受贿罪的帮助犯，但是是一个行为故想象竞合即可。

（二）利用影响力受贿罪

第三百八十八条之一【利用影响力受贿罪】
国家工作人员的近亲属或者其他与该国家工作人员关系密切的人，通过该国家工作人员职务上的行为，或者利用该国家工作人员职权或者地位形成的便利条件，通过其他国家工作人员职务上的行为，为请托人谋取不正当利益，索取请托人财物或者收受请托人财物，数额较大或者有其他较重情节的，处三年以下有期徒刑或者拘役，并处罚金；数额巨大或者有其他严重情节的，处三年以上七年以下有期徒刑，并处罚金；数额特别巨大或者有其他特别严重情节的，处七年以上有期徒刑，并处罚金或者没收财产。
离职的国家工作人员或者其近亲属以及其他与其关系密切的人，利用该离职的国家工作人员原职权或者地位形成的便利条件实施前款行为的，依照前款的规定定罪处罚。

请托人 A 向官员 C 的关系人 B 送钱，最终 B 向 C 说情等，B 构成利用影响力受贿。关系人 B 往往没有权力甚至不是国家工作人员，此时请托人 A 实际上是买 C 的权力，故 B 需要 C 许诺才能构成受贿罪。

区分与斡旋受贿罪：本罪中，中间人之所以能找终端官员办事是因为其能发挥影响力。而发挥影响力的根据在斡旋受贿中是因为工作关系“官官相护”。在利用影响力受贿罪中发挥关系是“私人关系”。但是两者在中间人包含多种身份关系时可以想象竞合。

  • 案例二
    A 对税务局长 C 的司机 B 行贿，B 许诺后对 C 说情，C 许诺。
    B 向 C 隐瞒收钱事实，C 只要许诺办事，B 就构成利用影响力受贿罪。
    C 收钱则直接构成普通受贿罪，B 构成受贿罪的共犯，同时构成利用影响力受贿罪，想象竞合。
    C 如果知情收钱事实，但没有收钱，C 依然是帮助他人收钱故构成普通受贿罪。B 依然是普通受贿罪共犯与利用影响力受贿罪。

（三）对有影响力的人行贿罪

第三百九十条之一【对有影响力的人行贿罪】
为谋取不正当利益，向国家工作人员的近亲属或者其他与该国家工作人员关系密切的人，或者向离职的国家工作人员或者其近亲属以及其他与其关系密切的人行贿的，处三年以下有期徒刑或者拘役，并处罚金；情节严重的，或者使国家利益遭受重大损失的，处三年以上七年以下有期徒刑，并处罚金；情节特别严重的，或者使国家利益遭受特别重大损失的，处七年以上十年以下有期徒刑，并处罚金。
单位犯前款罪的，对单位判处罚金，并对其直接负责的主管人员和其他直接责任人员，处三年以下有期徒刑或者拘役，并处罚金。

与上文模型相同，仅仅是对行贿方而言。

中间人的情况：请托人对于中间人要求将钱转交，如果中间人仅仅是转交的行为，则不构成斡旋受贿或者利用影响力受贿罪。如果是送给中间人钱，或者不明说时，则中间人会构成斡旋受贿罪或者利用影响力受贿罪

  1. 只要挪用公款就既遂。但是不一定处罚。客观处罚条件也是广义的犯罪成立条件。 ↩
  2. 受贿罪的实行行为不是办事行为，而是权钱交易行为，完全可以有事前事后两种方式。 ↩
  3. 此规定不合理之处在“对非国家工作人员行贿罪”中被动给钱不要求实际获得不正当利益，但是国家工作人员尚且要求，非国家工作人员更应当不要求，故“对非国家工作人员行贿罪”中即使是被动给钱也要求实际获得不正当利益。属于有利于被告人的类推解释。 ↩
